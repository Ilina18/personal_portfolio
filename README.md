# ilina-portfolio-full

A complete starter personal portfolio built with:
- Vite + React + TypeScript
- Tailwind CSS
- React Router (basic routes)

## What's included
- Home, Projects, About, Contact pages
- Navbar + Footer + Layout
- Project cards and placeholder assets

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run dev server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Notes
- Replace `hello@example.com` in `src/pages/Contact.tsx` with your real email.
- Add images into `src/assets/` and update `ProjectCard` image block.
